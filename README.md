## ✨ Overview

**Healthure** is an AI-powered healthcare platform that connects **patients** and **pharmacists** in one seamless ecosystem.

The platform helps patients manage medications, receive AI-powered medicine guidance, track medical records, and never miss important doses. Pharmacists can efficiently manage inventory, monitor sales, and improve patient safety through digital medication tracking.

# 🛠️ Tech Stack

| Technology | Purpose |
|------------|----------|
| **Next.js 15 (App Router)** | Frontend Framework |
| **Tailwind CSS** | Styling |
| **ShadCN UI** | UI Components |
| **Lucide React** | Icons |
| **Firebase Authentication** | User Authentication |
| **Firebase Firestore** | Database |
| **Google Genkit** | AI Integration |
| **Gemini 2.5 Flash** | Drug Analysis & AI Responses |
| **Text-to-Speech (TTS)** | Audio Medicine Summaries |
| **html5-qrcode** | QR & Barcode Scanning |
